# Cào việc làm báo lao động
1. Thu thập các bài viết của báo lao động gồm thông tin việc làm, ứng viên.
2. Tìm kiếm công việc phù hợp theo từ khóa.
3. Thống kê xem những ngành nghề nào đang hot.
